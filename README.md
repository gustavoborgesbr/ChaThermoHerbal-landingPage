# Thermo Detox Herbal

Página estática para divulgação e captura de leads do produto **Thermo Detox Herbal**.

## 🚀 Sobre o projeto
Esta landing page foi desenvolvida para:

- 🛡️ Apresentar produto com layout moderno e responsivo
- 🧪 Capturar leads via formulário integrado ao **Formspree** (sem backend)
- ⚡️ Oferecer performance otimizada (lazy loading, CSS moderno, animações leves)
- 🔍 Ser amigável para SEO (robots, sitemap, meta tags, schema.org)

## 🧩 Tecnologias usadas
- HTML + CSS (puro)
- JavaScript (Vanilla)
- Formspree (para capturar leads sem backend)

## 📁 Estrutura do repositório
- `index.html` – Página principal
- `css/styles.css` – Estilos principais
- `js/script.js` – Lógica de comportamento (nav, carrossel, formulário)
- `assets/` – Imagens usadas no site
- `robots.txt` / `sitemap.xml` – SEO
- `404.html` – Página de erro personalizada
- `DEPLOY_GUIDE.md` – Guia passo a passo para deploy

## ✅ Como usar
1. Clone o repositório
2. Substitua o Formspree ID em `js/script.js`
3. Faça deploy em Netlify/GitHub Pages/Vercel

## 📌 Observações
- Não é necessário backend (formulário funciona via Formspree)
- Recomendado usar HTTPS (Netlify/Vercel/GitHub Pages já oferecem)
